  OLD:

  2. Update sortOrder Assignment

  Replace this code:
  const tasksToCreate = categorizedIngredients.map((item, index) => {
    const categoryBase = CATEGORY_SORT_ORDER[item.category] || CATEGORY_SORT_ORDER['Other'];
    const sortOrder = categoryBase + index;

    return {

  With this:
  const tasksToCreate = categorizedIngredients.map((item, globalIndex) => {
    const categoryBase = CATEGORY_SORT_ORDER[item.category] || CATEGORY_SORT_ORDER['Other'];

    // Count how many items of this category we've seen so far
    const itemsOfCategoryBefore = categorizedIngredients
      .slice(0, globalIndex)
      .filter(i => i.category === item.category).length;

    const sortOrder = categoryBase + itemsOfCategoryBefore;

    return {
      title: ...,
      notes: ...,
      category: item.category,
      goalId: goal.id,
      completed: false,
      isToday: false,
      sortOrder
    };
  });

  Why This Matters

  Before: Items could get mixed up (e.g., "Other" items appearing between Produce items)

  After: All items of the same category stay together, sorted alphabetically within that category
